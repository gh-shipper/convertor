# ASCII Image & Video Translator
![Feature image](Converter.png)

A simple Python application for working with images and videos in a different format. The program allows users to load media files, adjust a few settings, and generate a modified output. It includes support for several file types and provides options for saving results.

## Features
* Works with common image and video formats
* Multiple conversion settings available
* Different display styles and color options
* Basic playback controls for videos
* Preview interface for viewing results
* Save outputs in various formats

## Setup
1. Install Python.
2. Install any required dependencies.
3. Run the application.

## Run
```bash
python ascii_translator.py
```

## Download (Windows)

Pre-built Windows executable (created with PyInstaller) can be placed in `dist/` as `ascii_translator.exe`. Use the website page at `website/index.html` to provide a one-click download link that points to `dist/ascii_translator.exe`.

To build locally:

```powershell
build_exe.bat
```

This will generate a Windows executable in `dist\ascii_translator.exe` using `Tabimage.ico` as the app icon.

## Usage
1. Open a file.
2. Adjust settings as needed.
3. Start the conversion process.
4. Save the generated result.
